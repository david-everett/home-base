# Finance Data

Personal expense tracking and budgeting data.

## Files

- `transactions.csv` — All logged expenses
- `budgets.json` — Monthly budget limits by category

## Categories

| Category | Description |
|----------|-------------|
| food | Groceries, lunch, snacks |
| dining | Restaurants, bars, delivery |
| transport | Uber, subway, gas |
| subscriptions | Streaming, software, memberships |
| shopping | Amazon, clothing, electronics |
| coffee | Coffee shops, beans |

## Logging Expenses

Add transactions manually or use the `log-expense` skill:
```
"Log $42.50 for lunch at Sweetgreen"
```

## Checking Spending

Use the `spending-summary` skill:
```
"How much have I spent on dining this month?"
"Am I on track for my food budget?"
```
